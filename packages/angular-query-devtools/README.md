# TanStack Angular Query Devtools

Developer tools for [@benjavicente/angular-query-experimental](https://www.npmjs.com/package/@benjavicente/angular-query-experimental).

See the [Angular Devtools documentation](https://tanstack.com/query/latest/docs/framework/angular/devtools).

## Installation

```bash
npm install @benjavicente/angular-query-devtools
```

Peer dependencies: `@angular/core`, `@angular/common`, and `@benjavicente/angular-query-experimental`.
